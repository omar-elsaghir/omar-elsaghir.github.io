#Omar Ramadan - Personal Portfolio

A personal portfolio website showcasing my work, skills, and services as a Data Engineer, AI Specialist, and Computer Science student. Built with a modern, dark-themed UI, focusing on clean code, responsive design, and smooth user interactions.

##🔗 Live Demo
[View Live Site](https://omar-ramadan.github.io/) *(Update this link once deployed to GitHub Pages)*

##✨ Features
* **Custom UI/UX:** Dark mode aesthetic with custom cursor tracking and noise overlay for a modern, tech-focused feel.
* **Responsive Design:** Fully responsive layout built with CSS Grid and Flexbox, ensuring a seamless experience across desktop and mobile devices.
* **Scroll Animations:** Integrated Intersection Observer API to trigger smooth fade-up animations as the user scrolls.
* **Project Showcase:** Interactive project cards linking directly to GitHub repositories.

##🛠️ Tech Stack
* **HTML5:** Semantic markup for structure and accessibility.
* **CSS3:** Custom variables, advanced selectors, animations (`@keyframes`), and custom scrollbars.
* **Vanilla JavaScript:** DOM manipulation for the custom cursor effect and scroll reveal logic.
* **Fonts:** [Syne](https://fonts.google.com/specimen/Syne) (Headings) & [IBM Plex Mono](https://fonts.google.com/specimen/IBM+Plex+Mono) (Body text).

##📂 Featured Projects
This portfolio highlights my core competencies in data engineering and analysis, including:
1. **8-Puzzle Solver:** An algorithmic implementation to solve the classic 8-puzzle game.
2. **ETL Pipeline for Student Records:** Automated data migration from Excel to SQL databases.
3. **SQL Data Analysis for E-commerce:** Extracting business insights, customer segmentation, and total sales analysis.
